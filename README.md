# 🚀 Exam Platform – Adaptive Online Assessment System

A full-stack, production-ready **online examination platform** designed to deliver secure, intelligent, and scalable assessments. The system combines adaptive testing, basic proctoring logic, performance analytics, and role-based access to simulate real-world certification environments.

---

## 🎯 Problem Statement

Traditional online exams lack adaptability, proper monitoring, and meaningful analytics. This platform addresses those gaps by providing:

* Adaptive question flow based on performance
* Basic cheating detection mechanisms
* Real-time progress tracking and analytics
* Role-based dashboards for admins and students

---

## ✨ Key Features

### 🧠 Adaptive Examination Engine

* Dynamically adjusts questions based on previous answers
* Stateful logic for personalized difficulty progression

---

### 🤖 AI-Inspired Features (Simulated)

* Mock question generation
* Performance insights with realistic delays
* Designed to simulate AI-driven systems

---

### 🛡️ Proctoring & Integrity System

* Tracks suspicious behavior during exams
* Automatically flags attempts when suspicion ≥ 50
* Logs all cheating-related events

---

### 🔐 Secure Attempt Control

* Prevents multiple concurrent attempts
* Only one active exam per student allowed

---

### 📊 Leaderboard System

* Ranking based on:

  * Score (descending)
  * Time taken (ascending)
  * Submission order

---

### 📄 Certificate Generation

* Generates certificates after completion
* PNG → PDF export
* Includes timestamps and user details

---

### 📝 Simplified Code Evaluation

* Basic validation (code length > threshold)
* Designed to simulate real coding assessments

---

### 📈 Dashboard & Analytics

* Track performance across exams
* View scores, attempts, and rankings

---

## 🏗️ Tech Stack

* **Frontend:** React, Tailwind CSS
* **Backend:** Node.js, Express
* **Database:** MongoDB Atlas
* **Authentication:** JWT-based auth
* **Deployment:** Vercel (Frontend), Render (Backend)

---

## 📂 Core Functionalities

* User Authentication (Login / Signup)
* Role-based access (Admin / Student)
* Exam creation and management
* Attempt tracking and persistence
* Real-time scoring and result processing
